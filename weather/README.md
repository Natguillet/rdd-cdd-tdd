# cli-weather

## Introduction

This application Web checks the weather for your city from your terminal.

## Pre-install
You need to install these packages to use the CLI.
- chalk
- lodash
- meow
- request
- update-notifier
- yql
```sh
❯ npm install meow
```
## Code Example
```sh
❯ node cli.j Paris France C
```
